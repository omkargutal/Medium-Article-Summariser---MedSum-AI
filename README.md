# 🧠 MedSum AI – Medium Article Summariser

An intelligent, AI-powered web app that transforms long Medium articles into concise, structured, and easy-to-digest summaries within seconds. Built for developers, readers, and knowledge seekers who value speed without losing context.

## ✨ Key Features

- **🔗 URL-Driven Summarization**: Simply paste any Medium article link to extract key insights instantly.

**sample articles for use**
```
https://medium.com/womenintechnology/stop-memorizing-design-patterns-use-this-decision-tree-instead-e84f22fca9fa

https://medium.com/@michalmalewicz/the-end-of-dashboards-and-design-systems-5d98ec9de627 
```

- **🧠 Advanced LLM Engines**: Leverages cutting-edge AI (OpenAI & local NLP) for high-quality, relevant summaries.
- **📄 Structured Content**: No more raw text dumps. Get your summaries in clean, readable sections.
- **🎯 Intelligent Noise Filtering**: Automatically strips out ads, newsletter popups, and irrelevant metadata.
- **⚡ Lightning-Fast Performance**: Optimized pipeline for rapid parsing and summarization.
- **📱 Responsive Design**: A distraction-free experience across all devices.

## 🚀 Getting Started

### 1. Clone the Repository
```bash
git clone https://github.com/omkargutal/Medium-Article-Summariser---MedSum-AI.git
cd Medium-Article-Summariser---MedSum-AI
```

### 2. Set Up Virtual Environment
```bash
python3 -m venv venv
source venv/bin/activate  # On Linux/Mac
venv\Scripts\activate # On Windows
```

### 3. Install Dependencies
```bash
pip install -r requirements.txt
```

### 4. Configuration
Create a `.env` file in the root directory and add your API credentials:
```env
OPENAI_API_KEY=your_openai_api_key_here
```

### 5. Launch the App
```bash
streamlit run app.py
```

## 📌 How to Use

1. **Enter URL**: Copy the link to any Medium article.
2. **Select Length**: Choose your desired summary length using the sidebar slider.
3. **Summarize**: Hit the "Summarize" button and watch the AI work its magic.
4. **Digest**: Read the structured summary and extract the value you need.
